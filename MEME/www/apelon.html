Policy Information
HOTLINE
 Hotline phone number page, call this number before 10:30AM Eastern for emergencies.
SOS Policy
 Documents the policy for handling SOS or emergency issues.  Use this tool to send a structured response to an SOS message from NLM.

Database Request Request database access.

 Online dubugging tool for server-side MEME errors.

 Manage MEME test suites

 Editor for MRFILES/MRCOLS data.
MRDOC Editor Editor for MRDOC data.
MRMAP Editor Viewer and Editor for Map Set data.
MRSAB Editor
 Editor for MRSAB data.

MEME On-line Tools
Action Harvester Retrieve/search action logs.
Activity Logs Review activity logs for recent database work.
Automated QA Manage Monster QA.  This tool allows the user to edit the checks used to QA the MID as well as run the script and review current/past logs of SRC and MID snapshots.
Change Password Use this tool to change your Jekyll/MEME password. If your password is already expired, a DBA can use this tool to change your password to something new.
Code Map Editor
 Manage various codes used by the editing system.  These codes are used for a vareity of different reasons, click here to see a list of valid code types and their descriptions.
Content View Editor
 Manage content views.
Integrity Editor
 Tool for editing the integrity system configuration, including adding/removing/updating checks, application vectors and editor override vectors.
MEME Properties Editor
 Edit metadata, for now only used to edit data for the DA attributes in MRSAT.
MID Maintenance
 Run common MID maintenance tasks, including research of database locks of MAX_TAB.
MID Validation Manage MID data validation.  This tool allows the user to edit the checks used to validate the MID as well as run the script and review current/past logs.
Release Maintenance
 Run common pre-production  maintenance tasks.
Research Unmapped Identifiers
 Used during source insertion if a "map to meme ids" step is unable to map all identifiers.  It finds exceptional cases that can be further researched.
SIMS Source Information Management System (SIMS). Note: This link is primarily for data entry use. To view the editor information, see the sources page which is generated by SIMS.
SRC Info Editor Edit information for MRSAB.
Termgroup Rank Editor
 Edit termgroup ranks in the MID.
"Why did this concept demote?" This is an investigative tool that acceps a concept id and tries to determine the cause of the demotion.  It finds the action that created the demotion, possible merge sets that were responsible and what failed integrity checks were the cause (with descriptions of the checks).
 
MRD On-line Tools

 Manage MRD data validation.  This tool allows the user to edit the checks used to validate the MRD as well as run the script and review current/past logs.
Release Manager Tool for building a release.
Termgroup Rank Editor
 Edit default release/MetamorphoSys termgroup rankings.
