Reporting problems with Source Content

For problems or errors with source content, editors should send an e-mail message to umlsmeta@nlm.nih.gov. This address is where NLM keeps an archive of problems with sources that can be reported at the appropriate time. Examples of items that can go to this address are:

Typos in the concept names from a source
Questions about meaning of concepts as defined by a source
Questions about relationships between concepts as defined by a source



If the source in question is MeSH, messages can be sent to meshsugg@nlm.nih.gov.

Editors should still edit the concepts that they have source questions about and edit them according to the source's meaning. Editors still need to edit the concept and not wait for an answer to their messages. Some of the issues may not be addressed until some point in the future. Note: If an editor thinks that the problems are the result of a data issue, they need to report it immediately to NLM per the instructions given for reporting MEME-II problems.


Remember that we have to represent the meaning in sources; and when the source has already published and used a version, it needs to be represented in Meta. We also have different relationships with different sources, which in turn have different abilities to accept and make corrections.
