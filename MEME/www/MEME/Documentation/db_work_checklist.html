 Checklist Items

Change made to core data

Major change made to core data (>100K rowS)
 Regenerate statistics
   exec dbms_stats.gather_schema_stats( -
        ownname => 'MTH', degree => 8, options => 'GATHER_STALE');
   exec dbms_stats.export_schema_stats('MTH','MTHSTATS');
 
  
  
ALWAYS
 Attempt to start MEME4 server on non-standard port to verify that data changes will not stop server from starting.
    $MEME_HOME/bin/start.pl -port 8085 -mid <mid> -log= &
    $MEME_HOME/bin/xreports.pl -c 100 -d <mid> -port 8085 -host localhost



TEST INSERTION
 Review the various merge logs and look for "bad" merges.  These will show up as "old" concept ids merging together where "old" means < min(concept_id) from source_concept_status.
