MEME_TESTER


 
This package is used to test or generate test data for other packages.

generate_batch_action_table 
    PROCEDURE generate_batch_action_table ( 
        action        IN VARCHAR2,
        id_type       IN VARCHAR2,
        sample_size   IN INTEGER,
        old_value     IN VARCHAR2 DEFAULT '',
        action_field  IN VARCHAR2 DEFAULT '');
This procedure generates a table which can be used to test the MEME_BATCH_ACTIONS package. The table names it produces are of the form: For example, calling generate_batch_action_table ('CS','C',100) would produce a table called: batch_cs_c_100.
generate_batch_insert_data 
    PROCEDURE generate_batch_insert_data ( 
        id_type       IN VARCHAR2,
        sample_size   IN INTEGER);
This procedure generates dummy data which can be used to test the MEME_BATCH_ACTIONS package.
exec_integrity_proc 
    PROCEDURE exec_integrity_proc;
This procedure executes MEME_INTEGRITY_PROC procedures for test purposes..
