Documentation for MEME4 Applications

Documentation is available for the following applications.
 
 Administrative Applications
Installing Software Info on the various "Installation Profiles". Details on how each system is isntalled and configured.
Starting/Stopping 
the Server		 Instructions on how to start (start.pl) and stop (shutdown.pl) the MEME Application Server.
Adding/Removing an Editor
 Documentation on adding/removing an editor.
Administrative Services Documentation on the perl client that provides a variety of administrative services from viewing the server log, to obtaining server statistics, to shutting down the server remotely.
Reporting Server Statistics Documentation on the perl client used to have the server report some interesting statistics (stats.pl).
Retrieving Server Log Documentation on the perl client used for retrieving the server log.
Running Client Applications Information on how to run applications that make use of the meme client api (memerun.pl).
Processing Medline Data Documentation on processes/tools for handling XML Medline baseline and udpate files.
MID Maintenanance Code samples and scripts for performing various MID maintenance operations.
Recurring Processes
 Detailing of processes that are run on a regular or semi-regular basis.
   
User Applications
Extended Concept Reports Documentation on the perl client used for generating extended concept reports (xreports.pl).
Old-style Concept Reports Documentation on the perl client used for generating concept reports (reports.pl).
MEME Rela Editor Kit Documentation for the quick implementation of molecular actions functionality needed for Vladimir's Rela Editor application.
Normalization Tools
 Documentation on the shell script wrappers for accessing norm, luiNorm, and wordind functions.
Printing HTML
 Documentation on the perl script wrapper for printing HTML or text documents in MEME4.
     
Source Insertion and Batch Action Applications
Batch Action Engine
 Documentation on the perl client used for performing batch actions  (batch.pl).
Merge Engine
 Documentation on the perl client used for running the merge engine  (merge.pl).
Batch Insertion
 Documentation on the perl client used for performing batch inserts  (insert.pl).
MM/TM Management
 Documentation on the perl client used for managing MTH/TM and MTH/MM atoms(mthtm.pl).
Stamping
 Documentation on the perl client used for stamping.
  
   
 DBA Applications
Reindex MID Documentation on the perl script used for reindexing the MID.
Rebuild MID Documentation on the perl script used for rebuilding MID.
Dump MID Documentation on the perl script used for writing MID tables to ASCII files.
