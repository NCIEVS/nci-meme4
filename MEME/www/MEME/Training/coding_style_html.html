Coding Style - HTML

This document contains descriptions of important sections of a Java class and is followed by a generic example.
 
General Rules
  • Use a program like Netscape Composer to write HTML pages so that you do not have to worry about the format.
  • Try to limit the length of lines in <pre> sections to < 80 characters so that it doesn't go too far off the page.
Footer Always have a footer like this:
 
<!-- These comments are used by the What's new Generator -->
<!-- Changed On: 2003/10/07 -->
<!-- Changed by: Brian Carlsen -->
<!-- Change Note: MEME - Training Documents - Coding style for HTML Documents -->
<!-- Fresh for: 1 month -->
 
Every time you update an HTML document, update the footer section to include the new date.
Version Information Track version information in the footer of each document, the version is merely the last changed date.
Environment and Defaults All HTML pages should eventually be installed on the web server.
