MRD_HOME Code Hierarchy

Code Hierarchy

Following is a brief description of $MRD_HOME and associated important code locations:

Location
 Description
$MRD_HOME This is the base directory for the MRD system.
$MRD_HOME/bin Useful executables for starting/stopping the server as well as managing some release processes.  Also the location of the server properties file.

Note: some of these scripts are ONLY called by MRD server components via java Runtime.exec() methods.
$MRD_HOME/cgi-bin
 CGI applications and log directories used by CGI applications
$MRD_HOME/doc Local javadoc and tables documentation
$MRD_HOME/log Location for logs.
$MRD_HOME/java Contains the class files (in a .jar) and the source files expanded in the subdirectories. A lvgapi.jar file should be linked to the current  /site/package/lvg/current/lib/lvg*api.jar file
$MRD_HOME/qa QA scripts (for validation)
$MRD_HOME/sql Schema scripts for tables, indexes, and PL/SQL packages
$MRD_HOME/webapps-mrd Web applications
/site/package/lvg/current Location of current LVG version.
/etc/umls/oracle.passwd Oracle password file (used for default connections)
/etc/umls/mid-services
 MID Services file
