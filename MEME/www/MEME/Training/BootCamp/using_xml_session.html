Goals

  • What you should learn:
    • Where XML is used during the Metathesaurus life cycle.
    • How we represent objects as XML documents.
  • What you should be able to do:
    • Use the ObjectXMLSerializer class to write an arbitrary Java object  to a file and read it back into an object.
