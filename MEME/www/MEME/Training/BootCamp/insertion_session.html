Lecture Outline

  1. Recipe Writing
    1. New source: new recipe
    2. Update source: update recipe
    3. Recipe Writer Java application
    4. The Anatomy of a Recipe (XML, HTML, or .csh)
      1. Load data
      2. Perform matching/merging
      3. Post-merge steps
    5. Future Recipe Writer Web application

  2. Source Insertion
    1. Preparation
      1. Receive .src files
      2. Create recipe
      3. Upload .src files and recipe .csh script
    2. Test insertion
      1. Perform test insertion
      2. Apelon review
      3. NLM review
      4. Redo test insertion if necessary
    3. Real insertion
      1. Perform real insertion
      2. Apelon review
      3. NLM review
      4. Perform patch operations if necessary
