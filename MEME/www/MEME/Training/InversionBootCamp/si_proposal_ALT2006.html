B.  Brief description of the input files.
 
==================================================================
 
ABC Code Short Description 2006.txt
 
This file contains all active ABC codes and their 28-character short descriptions. The file is tab-delimited. You will find the abbreviations used to create the 28-character short descriptions in the file "Abbreviations for ABC Short Descriptions 2006.txt".
 
CODE            5-alpha-character ABC code
SHORT_DESC_28   28-character short description of intervention
 
==================================================================
 
ABC Code Complete Description 2006.txt
 
This file contains all active ABC codes, their full-length hierarchical intervention descriptions and their expanded definitions. The file is tab-delimited.
 
CODE            5-alpha-character ABC code
CODE_DESC       Hierarchical full-length description of the 
                intervention
EXPANDED_DEF    More detailed description of the intervention that
                may include information about required training
                and certification
 
The "Long" descriptions are included in the "Complete" file.
The "Medium descriptions can be derived from the "Complete" file.
