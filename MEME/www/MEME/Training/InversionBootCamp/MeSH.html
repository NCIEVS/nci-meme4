Inversion Details of MeSH

Instructor

David Sherertz

Goals

  • What you should learn:
    • An understanding of MeSH .src files
  • What you should be able to do:
    • Recognize the MeSH.src files and understand their general purpose
    • Understand the process of going from MeSH XML to .src files
    • Comprehend where fields in the original MeSH XML files end up in .src files
    • Explain MeSH XML information that is omitted from the .src files

Materials

  • MeSH .src files Powerpoint Presentation
  • Auto-invert XML process documentation

Lecture Outline

  1. Overview of MeSH.src format
  2. Overview of how auto-invert scripting language works with MeSH XML files
  3. Highlights of MeSH.src files (classes_atoms, mergefacts, relationships, attributes)
  4. Other notable points about inverting MeSH(termgroups, sources, contexts)
  5. Summary and Discussion
