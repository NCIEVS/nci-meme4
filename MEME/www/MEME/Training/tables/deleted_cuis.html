deleted_cuis

Overview/Objective

This document contains a description of deleted_cuis and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:deleted_cuis
Table Type: not used
Description:Not yet being effectively used, this table tracks important information associated with deleted CUIs. Other tables manage mappings of deleted CUIs, but this table retains information such as what strings and sources at one time occupied that concept so that the CUI has some possibility for being resurrected in the future.
Fields:
code VARCHAR2(50) a value assigned to this atom/term by the source provider, this value has semantics specific to that source provider 
timestamp DATE indicates when this row was inserted or last modified 
cui VARCHAR2(10) a Concept Unique Identifier, e.g. C0000039 
lui VARCHAR2(10) must be in string_ui.lui 
isui VARCHAR2(10) must be in string_ui.isui 
aui VARCHAR2(10) must be in atoms_ui.aui 
root_source VARCHAR2(40) a value representing a source independent of version which can be used across time to mean the same source 
Indexes: No indexes
