editor_preferences

Overview/Objective

This document contains a description of editor_preferences and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:editor_preferences
Table Type: metadata
Description:Tracks preference settings used by the editing interface. In particular, it allows users to specify which frames should open when the interface is started
Fields:
name VARCHAR2(9) the name of a qa check 
initials VARCHAR2(3) 3-letter initials used by an editor 
show_concept NUMBER(12) a 1/0 flag indicating a users preference to auto-open the concept frame in the interface 
show_classes NUMBER(12) a 1/0 flag indicating a users preference to auto-open the classes frame in the interface 
show_relationships NUMBER(12) a 1/0 flag indicating a users preference to auto-open the relationships frame in the interface 
show_attributes NUMBER(12) a 1/0 flag indicating a users preference to auto-open the attributes frame in the interface 
Indexes:
  • editor_preferences_pk ON name
