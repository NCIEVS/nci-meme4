mom_candidate_facts

Overview/Objective

This document contains a description of mom_candidate_facts and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:mom_candidate_facts
Table Type: source
Description:Contains facts being considered for merging. The processes that load data from mergefacts.src files or generate facts from the database put those candidate facts here.
Fields:
atom_id_1 NUMBER(12) must be in classes.atom_id or in dead_classes.atom_id 
atom_id_2 NUMBER(12) must be in classes.atom_id or in dead_classes.atom_id 
code1 VARCHAR2(30) the code of the atom_id_1 
code2 VARCHAR2(30) the code of the atom_id_2 
termgroup1 VARCHAR2(40) the termgroup of the atom_id_1 
termgroup2 VARCHAR2(40) the termgroup of the atom_id_2 
source1 VARCHAR2(20) the source of the atom_id_1 
source2 VARCHAR2(20) the source of the atom_id_2 
merge_level VARCHAR2(3) a value indicating the type of merge fact, usually one of MAT (exact match), NRM (norm match), or SY (synonym) 
status VARCHAR2(1) not used 
lui_1 VARCHAR2(10) the lui of the atom_id_1 
lui_2 VARCHAR2(10) the lui of the atom_id_2 
isui_1 VARCHAR2(10) the isui of the atom_id_1 
isui_2 VARCHAR2(10) the isui of the atom_id_2 
sui_1 VARCHAR2(10) the sui of the atom_id_1 
sui_2 VARCHAR2(10) the sui of the atom_id_2 
merge_set VARCHAR2(30) the name of the merge set that this fact belongs to 
source VARCHAR2(20) the source of this fact 
Indexes: No indexes
Insertion Role:This table is used by the merge engine as a staging area for merge facts to process. This table is loaded by either the MEME_SOURCE_PROCESSING.prepare_src_mergefacts or by MEME_SOURCE_PROCESSING.generate_facts
