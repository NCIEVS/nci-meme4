content_view_members

Overview/Objective

This document contains a description of content_view_members and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:content_view_members
Table Type: auxiliary
Description:Contains Metathesaurus UIs (AUI, RUI, ATUI, CUI) indicating membership in a content view. Links back to content_views via the content_view_id field.
Fields:
meta_ui VARCHAR2(20) a Metathesaurus identifier (AUI, RUI, ATUI, CUI) 
content_view_id NUMBER(12) unique identifier for a content view 
Indexes:
  • x_cvm_mui ON meta_ui
    • Production Role:Source of information for the CVF fields of the various content files.
