MEME Tables - sg_molecular_actions

Following is a description of sg_molecular_actions and its fields
 

Table name:sg_molecular_actions
Description:Not currently used. This table was envisioned as a parallel to molecular_actions for integrated vocabulary development (editing of sources within the MID).
Fields:
transaction_id NUMBER(12) must be in molecular_actions.transaction_id 
molecule_id NUMBER(12) a unique identifier for each row, primary key 
authority VARCHAR2(20) the authority responsible for this row, can be validated by MEME_RANKS.get_source_authority_rank 
timestamp DATE indicates when this row was inserted or last modified 
molecular_action VARCHAR2(30) a code indicating the action performed, must be in code_map where the type='molecular_action' 
source_sg_id VARCHAR2(50) a generic type if identifier, originally for source group identifier 
source_sg_type VARCHAR2(50) a field indicating the type of an sg_id, valid values can be found in code_map.code where the type='sg_type' 
source_sg_qualifier VARCHAR2(50) some sg_types require two pieces of information, an id and a qualifier, e.g. for CODE_SOURCE it would be a source value 
target_sg_id VARCHAR2(50) a generic type if identifier, originally for source group identifier 
target_sg_type VARCHAR2(50) a field indicating the type of an sg_id, valid values can be found in code_map.code where the type='sg_type' 
target_sg_qualifier VARCHAR2(50) some sg_types require two pieces of information, an id and a qualifier, e.g. for CODE_SOURCE it would be a source value 
undone VARCHAR2(1) a Y/N flag indicating whether or not this action was undone 
undone_by VARCHAR2(20) the authority responsible for undoing this action, if undone='Y' 
undone_when DATE the timestamp of the undoing of this action, if undone='Y' 
status VARCHAR2(1) must be in level_status_rank.status where the table_name matches the code for this core table 
elapsed_time NUMBER(12) an integer number of seconds 
work_id NUMBER(12) must be either 0 or must be in meme_work.work_id 
Indexes:
  • sg_molecular_actions_pk ON molecule_id
  • x_sgma_sid ON source_sg_id
  • x_sgma_tid ON target_sg_id
