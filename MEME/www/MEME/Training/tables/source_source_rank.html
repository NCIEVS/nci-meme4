source_source_rank

Overview/Objective

This document contains a description of source_source_rank and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:source_source_rank
Table Type: source
Description:Staging area for new sources.
Fields:
high_source VARCHAR2(20) a new or update source to add to  
low_source VARCHAR2(20) the source immediately above which the high_source should be ranked 
restriction_level NUMBER(12) a 0-3 value indicating what type of licensing arrangement is necessary to use this source 
normalized_source VARCHAR2(20) used to map a source value that does not have an SRC concept to one that does, e.g. MSH2002HMCE has a normalized source of MSH2002 
stripped_source VARCHAR2(20) a value representing a source independent of version which can be used across time to mean the same source 
source_official_name VARCHAR2(3000) the SRC/VPT name for this source 
source_family VARCHAR2(20) a value representing a group of sources that are connected to one another, e.g. CPT and MTHCH have the same source family: CPT 
version VARCHAR2(20) the version number of a component/application 
valid_start_date DATE the date that the source becomes valid 
valid_end_date DATE the date that the source ceases to be valid 
insert_meta_version VARCHAR2(20) the version of the Metathesaurus in which the source first appears, e.g. 2001AA 
remove_meta_version VARCHAR2(20) the version of the Metathesaurus in which the source is removed, e.g. 2002AA 
nlm_contact VARCHAR2(100) the person at NLM responsible for this source 
inverter_contact VARCHAR2(100)  
acquisition_contact VARCHAR2(1000) contact information for the person from whom the source can be obtained 
content_contact VARCHAR2(1000) contact information for the source provider person who can best answer content questions 
license_contact VARCHAR2(1000) contact information for the source provider person who can best answer licensing questions 
release_url_list VARCHAR2(1000) a list of URLs related to this source, for release to the public 
context_type VARCHAR2(100) a value indicating what type of contexts this source has, according to section 2.3.2 of the documentation 
language VARCHAR2(10) the name of a language 
citation VARCHAR2(4000) citation information 
license_info VARCHAR2(4000) information about obtaining a license for this source 
character_set VARCHAR2(50) an ISO character set name 
rel_directionality_flag VARCHAR2(1) a Y/N flag indicating whether or not this source asserts the direction of its relationships 
rank NUMBER(12) the rank of this row, higher values are better ranks 
Indexes: No indexes
Insertion Role:First, they are loaded from sources.src, then the sources are prepared for insertion and finally they are loaded into source_rank.
