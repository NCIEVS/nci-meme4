src_qa_results

Overview/Objective

This document contains a description of src_qa_results and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:src_qa_results
Table Type: qa
Description:A record of the last Monster QA run against the source core tables. Click here for Automated QA System documentation.
Fields:
qa_id NUMBER(12) a unique id given to a Monster QA set 
name VARCHAR2(100) the name of a qa check 
value VARCHAR2(100) a value 
qa_count NUMBER(12) the count associated with a particular qa check 
timestamp DATE indicates when this row was inserted or last modified 
Indexes: No indexes
Editing Role:These results can be compared against the MID results in mid_qa_results by using the Automated QA system.
