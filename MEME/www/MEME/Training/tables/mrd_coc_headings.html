mrd_coc_headings

Overview/Objective

This document contains a description of mrd_coc_headings and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:mrd_coc_headings
Table Type: unknown
Description:This table is part of a two-table representation of co-occurrence data in the MRD. States are added/expired whenever the COC data changes.
Fields:
citation_set_id NUMBER(12) an id assigned to represent a set of citations for an article within a particular source such as Medline 
publication_date DATE the date of publication of an article from which citations are drawn 
heading_aui VARCHAR2(10)  
major_topic VARCHAR2(1) a Y/N flag indicating whether this subheading are considered major topics, this corresponds to a *'d entry in the old Medline file format 
subheading_set_id NUMBER(10) an unique id within the citation_set_id identifying (groups of) subheadings 
root_source VARCHAR2(20) a value representing a source independent of version which can be used across time to mean the same source 
coc_type VARCHAR2(10) indicates the nature of a citation and what type of co-ocurrence it may become. The values are those from the MRCOC.COT field 
insertion_date DATE indicates the date this row was inserted 
expiration_date DATE  
Indexes:
  • x_mcoch_cid ON citation_set_id
  • x_mcoch_haui ON heading_aui
