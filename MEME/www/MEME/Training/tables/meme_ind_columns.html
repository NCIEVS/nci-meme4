meme_ind_columns

Overview/Objective

This document contains a description of meme_ind_columns and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:meme_ind_columns
Table Type: system
Description:A projection of user_ind_columns that tracks index columns for MEME tables
Fields:
index_name VARCHAR2(30) taken from user_ind_columns 
table_name VARCHAR2(30) taken from user_ind_columns 
column_name VARCHAR2(4000) taken from user_ind_columns 
column_position NUMBER() taken from user_ind_columns 
column_length NUMBER() taken from user_ind_columns 
descend VARCHAR2(4) taken from user_ind_columns 
Indexes: No indexes
Editing Role:During the editing cycle, the core tables are reindexed daily and the structure of those indexes is taken from this table and from meme_indexes. The various reindexing procedures in MEME_SYSTEM make use of this table.
