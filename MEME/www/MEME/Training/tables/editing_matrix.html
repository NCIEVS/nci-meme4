editing_matrix

Overview/Objective

This document contains a description of editing_matrix and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:editing_matrix
Table Type: qa
Description:Tracks aggregate status information for a concept. This table is used primarily by the matrix initializer to aggregate status information about a concepts classes, attributes, and relationships. Furthermore, it tracks any integrity violations that are found at the time of the matrix calculation.
Fields:
concept_id NUMBER(12) must be in classes.concept_id 
classes_status VARCHAR2(1) an indicator of the status of all atoms/terms within a concept 
attributes_status VARCHAR2(1) an indicator of the status of all attributes within a concept 
relationships_status VARCHAR2(1) an indicator of the status of all relationships within a concept 
integrity_status VARCHAR2(1) an indicator of the overall integrity status of a concept 
integrity_vector VARCHAR2(2000) a vector of violations of integrity checks , e.g. <DT_M1:V><DT_PN1:W> 
clean_molecule_id NUMBER(12) the id of the last action which left this concept in a clean state, must be in molecular_actions.molecule_id 
last_app_change DATE the timestamp of the last approved action 
last_app_change_by VARCHAR2(50) the authority of the last approved action 
last_unapp_change DATE not used 
last_unapp_change_by VARCHAR2(50) not used 
concept_status VARCHAR2(1) the overall status of the concept, used to determine if the concept needs to be edited 
Indexes:
  • editing_matrix_pk ON concept_id
    • Editing Role:The matrix initializer is run daily to determine which concepts still need editing. The results of its run are (partially) stored in this table, including the aggregated status of the atoms, attributes, and relationships as well as the overall integrity state for each concept. Concepts that require editing of any of their core elements or require editing for integrity reasons are unapproved by this process. Generally, if one wants to know why something was unapproved, this table can provide the answer.
