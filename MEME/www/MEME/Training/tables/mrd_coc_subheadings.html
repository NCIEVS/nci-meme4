mrd_coc_subheadings

Overview/Objective

This document contains a description of mrd_coc_subheadings and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:mrd_coc_subheadings
Table Type: unknown
Description:This table is the second part of the two-table representation of co-occurrence data in the MRD. Headings can co-occur in a citation and have certain subheadings connected to them. Those subheading relationships are reprsented here.
Fields:
citation_set_id NUMBER(12) an id assigned to represent a set of citations for an article within a particular source such as Medline 
subheading_set_id NUMBER(10) an unique id within the citation_set_id identifying (groups of) subheadings 
subheading_qa CHAR a two-letter abbreviation of a MeSH qualifier, must be in attributes.attribute_value where the attribute_name='QA' 
subheading_major_topic VARCHAR2(1) a Y/N flag indicating whether this heading or any of its subheadings are considered major topics, this corresponds to a starred (*) entry in the old Medline file format 
insertion_date DATE indicates the date this row was inserted 
expiration_date DATE  
Indexes:
  • x_mcocsh_ssid ON citation_set_id
