meme_tables

Overview/Objective

This document contains a description of meme_tables and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:meme_tables
Table Type: system
Description:A list of tables maintained by the MEME system. Tables in this list generally are created by the /sql/tables.sql schema script and have their indexes maintained in the meme_indexes and meme_ind_columns tables. In the future this table will likely track additional information such as what type of table each entry is and what the formal update model for those tables should be.
Fields:
table_name VARCHAR2(50) a name of a table in the MID that is tracked, must be in user_tables.table_name 
Indexes: No indexes
Editing Role:The various mid-wide system management procedures in MEME_SYSTEM make use of this table (e.g. MEME_SYSTEM.analyze_mid.
