allocated_ui_ranges

Overview/Objective

This document contains a description of allocated_ui_ranges and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:allocated_ui_ranges
Table Type: not used
Description:Tracks ranges of UIs allocated for certain reasons. These UIs can be of any type, but will likely be things like SUIs, LUIs, and AUIs.
Fields:
type VARCHAR2(50) a value indicating the type of identifier that has been allocated 
low_ui VARCHAR2(10) the lower bound of a unique identifier range 
high_ui VARCHAR2(10) the upper bound of a unique identifier range 
root_source VARCHAR2(20) a value representing a source independent of version which can be used across time to mean the same source 
description VARCHAR2(2000) a detailed description 
Indexes: No indexes
