system_status

Overview/Objective

This document contains a description of system_status and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:system_status
Table Type: system
Description:Generic mechanism for applications that need an ON/OFF switch. Specifically, it aggregates the functionality of dba_cutoff and ic_system_status which are now views into this table.
Fields:
system VARCHAR2(1000) a label indicating a switch of some kind 
status VARCHAR2(100) the status of the switch corresponding to the system field 
Indexes: No indexes
