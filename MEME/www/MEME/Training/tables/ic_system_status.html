MEME Tables - ic_system_status

Following is a description of ic_system_status and its fields
 

Table name:ic_system_status
Description:Acts like a switch for the integrity system. If the switch is set to OFF checks are ignored during editing actions. If the switch is set to ON checks are performed according to the integrity vectors used. If the switch is W checks are all performed with ic_code of W.
Fields:
status VARCHAR2(3) one of ON, OFF, or W (warning mode) 
version NUMBER(12) the version number of a component/application 
Indexes: No indexes
