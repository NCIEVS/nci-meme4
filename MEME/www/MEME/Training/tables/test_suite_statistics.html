test_suite_statistics

Overview/Objective

This document contains a description of test_suite_statistics and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:test_suite_statistics
Table Type: system
Description:Stores statistics gathered during test suite runs.
Fields:
set_name VARCHAR2(100) the name of the test suite set 
suite_name VARCHAR2(100) the name of the test suite 
test_name VARCHAR2(100) the name of the test 
value VARCHAR2(4000) the value generated by this test 
timestamp DATE indicates when this row was inserted or last modified 
elapsed_time NUMBER(12) an integer number of milliseconds 
authority VARCHAR2(50) the authority responsible for this row, can be validated by MEME_RANKS.get_source_authority_rank 
error VARCHAR2(4000) the error details, if an error occurred while running this test 
Indexes: No indexes
