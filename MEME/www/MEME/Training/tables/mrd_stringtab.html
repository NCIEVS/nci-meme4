mrd_stringtab

Overview/Objective

This document contains a description of mrd_stringtab and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:mrd_stringtab
Table Type: unknown
Description:This table represents mrd long attribute valuestates. This is basically a historical version of stringtab.
Fields:
string_id NUMBER(12) an id that groups fragments of a long string together 
row_sequence NUMBER(12) a positive integer used for ordering rows of the table 
text_total NUMBER(12) the length of a string when all of the fragments are put together 
text_value VARCHAR2(1786) one fragment of a potentially larger string, bounded to 1786 by OpenROAD 
insertion_date DATE indicates the date this row was inserted 
expiration_date DATE  
Indexes:
  • x_mstab_sid ON string_id, row_sequence
