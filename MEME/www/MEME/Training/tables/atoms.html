atoms

Overview/Objective

This document contains a description of atoms and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:atoms
Table Type: auxiliary
Description:Holds atom names for each atom. This table is redundant because the data is in string_ui.string.
Fields:
atom_id NUMBER(12) must be in classes.atom_id 
version_id NUMBER(12) not used 
atom_name VARCHAR2(3000) the actual name associated with an atom/term, must be in string_ui.string 
Indexes:
  • atoms_pk ON atom_id
  • x_atoms_an ON atom_name
    • Insertion Role:Atom names are loaded into this table by MEME_BATCH_ACTIONS.macro_insert.
