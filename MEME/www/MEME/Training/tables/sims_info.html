sims_info

Overview/Objective

This document contains a description of sims_info and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:sims_info
Table Type: metadata,source
Description:Tracks source metadata. There is a lot of information about sources in this table, much of it details the inversion and insertion process.
Fields:
whats_new VARCHAR2(4000)  
source VARCHAR2(20) must be in source_rank.source 
date_created DATE  
meta_year VARCHAR2(20)  
init_rcpt_date DATE  
clean_rcpt_date DATE  
test_insert_date DATE  
real_insert_date DATE  
source_contact VARCHAR2(4000)  
inverter_contact VARCHAR2(100)  
nlm_path VARCHAR2(200)  
apelon_path VARCHAR2(200)  
inversion_script VARCHAR2(100)  
inverter_notes_file VARCHAR2(100)  
conserve_file VARCHAR2(100)  
sab_list VARCHAR2(100)  
meow_display_name VARCHAR2(4000)  
source_desc VARCHAR2(4000)  
status VARCHAR2(10) must be in level_status_rank.status where the table_name matches the code for this core table 
worklist_sortkey_loc VARCHAR2(100)  
termgroup_list VARCHAR2(4000)  
attribute_list VARCHAR2(4000)  
inversion_notes VARCHAR2(4000)  
notes VARCHAR2(4000) any additional information 
inv_recipe_loc VARCHAR2(100)  
suppress_edit_rec VARCHAR2(1)  
versioned_cui VARCHAR2(10) the CUI assigned to the root SRC concept representing this source in the last release 
root_cui VARCHAR2(10) the CUI assigned to the root SRC concept representing this source in the last release 
source_official_name VARCHAR2(4000) the SRC/VPT name for this source 
source_short_name VARCHAR2(4000) the short name (used by UMLSKS) for this source 
attribute_name_list VARCHAR2(4000) a comma separated list of ATN values 
term_type_list VARCHAR2(1000) a comma separated list of TTY values 
term_frequency NUMBER(12) CUI,SUI frequency of this source in the last release 
cui_frequency NUMBER(12) CUI frequency of this source in the last release 
citation VARCHAR2(4000) citation information 
last_contacted DATE the date the source provider was last contacted regarding this source 
license_info VARCHAR2(4000) information about obtaining a license for this source 
character_set VARCHAR2(50) an ISO character set name 
valid_start_date DATE the date that the source becomes valid 
valid_end_date DATE the date that the source ceases to be valid 
insert_meta_version VARCHAR2(20) the version of the Metathesaurus in which the source first appears, e.g. 2001AA 
remove_meta_version VARCHAR2(20) the version of the Metathesaurus in which the source is removed, e.g. 2002AA 
nlm_contact VARCHAR2(100) the person at NLM responsible for this source 
acquisition_contact VARCHAR2(4000) contact information for the person from whom the source can be obtained 
content_contact VARCHAR2(4000) contact information for the source provider person who can best answer content questions 
license_contact VARCHAR2(4000) contact information for the source provider person who can best answer licensing questions 
context_type VARCHAR2(100) a value indicating what type of contexts this source has, according to section 2.3.2 of the documentation 
language VARCHAR2(10) the name of a language 
test_insertion_start DATE the start date of a the test insertion for this source 
test_insertion_end DATE the end date of a the test insertion for this source 
real_insertion_start DATE the start date of a the real insertion for this source 
real_insertion_end DATE the end date of a the real insertion for this source 
editing_start DATE the date when editing of this source started 
editing_end DATE the date when editing of this source was completed 
latest_available VARCHAR2(100) the most recent version of the source 
release_url_list VARCHAR2(4000) a list of URLs related to this source, for release to the public 
internal_url_list VARCHAR2(4000) a list of URLs related to this source, for internal consumption 
rel_directionality_flag VARCHAR2(1) a Y/N flag indicating whether or not this source asserts the direction of its relationships 
Indexes: No indexes
Insertion Role:This table is populated manually during the inversion through the use of SIMS. Additional information is added during the insertion process from sources.src. 
Production Role:Source of information for MRSAB.
