dead_foreign_classes

Overview/Objective

This document contains a description of dead_foreign_classes and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:dead_foreign_classes
Table Type: auxiliary
Description:Container for deleted foreign atoms.
Fields:
atom_id NUMBER(12) must be in classes.atom_id or in dead_classes.atom_id 
eng_atom_id NUMBER(12) Pointer to the the ENG (English) atom corresponding to this row. Must be in classes.atom_id. 
eng_aui VARCHAR2(10)  
language VARCHAR2(10) the name of a language 
version_id NUMBER(12) not used 
source VARCHAR2(20) must be in source_rank.source 
termgroup VARCHAR2(40) must be in termgroup_rank.termgroup 
tty VARCHAR2(20) the term type, must be in termgroup_rank.tty 
termgroup_rank NUMBER(12) comes from termgroup_rank.rank where the termgroup fields match 
code VARCHAR2(30) a value assigned to this atom/term by the source provider, this value has semantics specific to that source provider 
sui VARCHAR2(10) must be in string_ui.sui 
lui VARCHAR2(10) must be in string_ui.lui 
generated_status VARCHAR2(1) must be in ('Y','N') 
last_release_cui VARCHAR2(10) the cui of this atom/term in the last (major) release 
dead VARCHAR2(1) a Y/N flag indicating whether or not this row is dead 
status VARCHAR2(1) must be in level_status_rank.status where the table_name matches the code for this core table 
authority VARCHAR2(50) the authority responsible for this row, can be validated by MEME_RANKS.get_source_authority_rank 
timestamp DATE indicates when this row was inserted or last modified 
insertion_date DATE indicates the date this row was inserted 
concept_id NUMBER(12) must be in concept_status.concept_id or in dead_concept_status.concept_id 
released VARCHAR2(1) must be in released_rank.released 
tobereleased VARCHAR2(1) must be in tobereleased_rank.tobereleased 
last_molecule_id NUMBER(12) must be in molecular_actions.molecule_id and is the id of the last molecular action to affect this core table row 
last_atomic_action_id NUMBER(12) must be in atomic_actions.atomic_action_id and is the id of the last atomic action to affect this core table row 
sort_key NUMBER(12) not used 
rank NUMBER() the rank of this row, higher values are better ranks 
last_release_rank NUMBER(12) a 0-4 value indicating the TS,Stt rank of this row in the last (major) release.
  • 4: P|PF
  • 3: S|PF
  • 2: P|V*
  • 1: S|V*
  • 0: not in previous release
 
suppressible VARCHAR2(10) must be in suppressible_rank.suppressible 
last_assigned_cui VARCHAR2(10) the last cui assigned to this atom/term by the most recent cui assignment run 
isui VARCHAR2(10) must be in string_ui.isui 
aui VARCHAR2(10) must be in atoms_ui.aui 
source_aui VARCHAR2(50) a source asserted unique id for each atom as it appears in the native source files (e.g. SNOMED DESCRIPTIONID) 
source_cui VARCHAR2(50) a source asserted unique id for each concept as it appears in the native source files (e.g. SNOMED CONCEPTID) 
source_dui VARCHAR2(50) a source asserted unique id for each descriptor as it appears in the native source files (e.g. MeSH D#) 
Indexes:
  • d_f_classes_pk ON atom_id
