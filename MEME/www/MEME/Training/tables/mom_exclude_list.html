mom_exclude_list

Overview/Objective

This document contains a description of mom_exclude_list and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:mom_exclude_list
Table Type: source
Description:A list of termgroups used as a filter when generating some merge facts. Typically the termgroups on this list are abbreviations.
Fields:
termgroup VARCHAR2(40) must be in termgroup_rank.termgroup 
Indexes: No indexes
Insertion Role:Used by MEME_SOURCE_PROCESSING.filter_facts to remove merge facts from consideration which have termgroups on the list.
