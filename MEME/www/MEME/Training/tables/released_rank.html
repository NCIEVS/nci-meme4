released_rank

Overview/Objective

This document contains a description of released_rank and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:released_rank
Table Type: auxiliary,metadata
Description:Normalizes and ranks values of released fields in the various core tables.
Fields:
released VARCHAR2(1) must be in released_rank.released 
rank NUMBER(12) the rank of this row, higher values are better ranks 
Indexes: No indexes
Editing Role:When any of the MEME_APROCS.aproc_insert procedures are called, this table is used to validate the released field
