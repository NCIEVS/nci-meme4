normwrd

Overview/Objective

This document contains a description of normwrd and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:normwrd
Table Type: index
Description:A normalized word index. Looking up a normalized word in here will yield a set of atoms whose normalized string contains that word.
Fields:
normwrd_id NUMBER(12) must be in classes.atom_id or in dead_classes.atom_id 
normwrd VARCHAR2(100) a word taken from a normalized string by the LVG wordind program 
Indexes:
  • x_nw_nw ON normwrd
  • x_nw_nwid ON normwrd_id
    • Editing Role:During the editing cycle, this table is used by the Finder to perform normalized word lookups in the classes table.
