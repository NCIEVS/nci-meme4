source_coc_headings_todelete

Overview/Objective

This document contains a description of source_coc_headings_todelete and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:source_coc_headings_todelete
Table Type: unknown
Description:Staging area for deleted heading citation set information.
Fields:
citation_set_id NUMBER(12) an id assigned to represent a set of citations for an article within a particular source such as Medline 
Indexes: No indexes
Insertion Role:When updating coc heading information from Medline update files (/bin/update_medline_data.csh) citation set ids to delete are loaded into this table by the /bin/process_medline_data.csh script, then removed from the coc_headings and coc_subheadings tables.
