code_map

Overview/Objective

This document contains a description of code_map and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:code_map
Table Type: metadata
Description:Contains a variety of useful mappings. This table is used for mapping almost every kind of code in the MEME system to a relevant value. To see the types of codes mapped, look up the rows where the type='valid_code_type'.
Fields:
code VARCHAR2(50) a value assigned to this atom/term by the source provider, this value has semantics specific to that source provider 
type VARCHAR2(50) a code type, must be in code_map.code where type='valid_code_type' 
value VARCHAR2(4000) the value that a code is mapped to 
Indexes:
  • code_map_pk ON code, type
