qa_diff_adjustment

Overview/Objective

This document contains a description of qa_diff_adjustment and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:qa_diff_adjustment
Table Type: qa
Description:Tracks adjustments to Monster QA discrepancies between current and historical MID counts. Click here for Automated QA System documentation.
Fields:
qa_id_1 NUMBER(12) a unique id given to a Monster QA set 
qa_id_2 NUMBER(12) a unique id given to a Monster QA set 
name VARCHAR2(100) the name of a qa check 
value VARCHAR2(100) a value 
qa_count NUMBER(12) the adjustment value between the sets defined by qa_id_1 and qa_id_2 
timestamp DATE indicates when this row was inserted or last modified 
description VARCHAR2(1000) a detailed description 
Indexes: No indexes
Editing Role:Used by the Automated QA system to adjust counts that are off for known reasons.
