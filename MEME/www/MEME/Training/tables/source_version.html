source_version

Overview/Objective

This document contains a description of source_version and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:source_version
Table Type: auxiliary,metadata
Description:Maintains a list of current and previous versions of sources. These values can be looked up by querying the table with a stripped source value.
Fields:
source VARCHAR2(20) must be in source_rank.stripped_source 
current_name VARCHAR2(20) the name of the current version of a source in the Metathesaurus 
previous_name VARCHAR2(20) the name of the immediately previous version of a source that has been updated 
Indexes: No indexes
Insertion Role:When inserting an update source, this table is updated to reflect the current/previous versions of that source. 
Production Role:This table is used to determine what the current version of each source is.
