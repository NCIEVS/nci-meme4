source_coc_headings

Overview/Objective

This document contains a description of source_coc_headings and its various fields. It also (may) include information about how this table is used in the various stages of MID processing.
 

Details

Table name:source_coc_headings
Table Type: source
Description:Staging area for new heading citations.
Fields:
citation_set_id NUMBER(12) an id assigned to represent a set of citations for an article within a particular source such as Medline 
publication_date DATE the date of publication of an article from which citations are drawn 
heading_id NUMBER(12) an id representing an atom/term that was found as a citation in some article, must be in classes.atom_id 
major_topic VARCHAR2(1) a Y/N flag indicating whether this subheading are considered major topics, this corresponds to a *'d entry in the old Medline file format 
subheading_set_id NUMBER(10) an unique id within the citation_set_id identifying (groups of) subheadings 
source VARCHAR2(20) must be in source_rank.source 
coc_type VARCHAR2(10) indicates the nature of a citation and what type of co-ocurrence it may become. The values are those from the MRCOC.COT field 
Indexes: No indexes
Insertion Role:During an insertion, citation information is loaded into this table by the /bin/process_medline_data.csh script, then it is loaded into coc_headings.
