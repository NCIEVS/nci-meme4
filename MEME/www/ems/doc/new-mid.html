Starting a New MID

  • Once editing has been frozen in preparation for a release, Apelon, coordinating with NLM, will create and initialize a new MID on the production machine. The EMS tables (all tables with name like ems_% or wms_% in the meow schema) should have been copied over (Were they?).

  • Once the DB is ready, start a new editing epoch as described here.

  • Once that is done, editing can begin after a matrix init and a repartition assuming you have configured the ME bins, QA bins, etc. as needed.
