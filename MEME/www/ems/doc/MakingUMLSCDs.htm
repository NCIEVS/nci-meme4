Burning CDs for the UMLS Releases
-Run Adaptec Easy CD Creator 4.x
-Select the files that you wish to place on the CD from the top half of the window.
    These files include the TGZ or the ZIP files containing the UMLS Metathesaurus MR files.
    Drag these files to the CD Layout portion of the window.

-Select "Add items to CD layout" from the popup menu

-From the Main Menu select File | Properties

o Volume Label: Enter volume label text (i.e. 2001_TAR)

o File System: "ISO9660"

o Select: "Preserve normal file ordering"

o Check: "Single data track only"

o Select: "Mode 1: CDROM"

-Click on the File Systems tab

o Select: "Use this date" and enter in the time/date stamp you wish to use.

-Select OK to close the File Properties dialog box

-From the Main Menu select File | Create CD

o Be sure that you are writing to the correct device

o Verify the write speed against the specs for your writer

o Set number of copies (usually 1)

-Click on the "Advanced >>" button

-Under "Create options" select

o "Test and Create CD" or just "Create CD" if you are have tested the CD writer out already

-Under the "Write Method" select

o Select: "Track-at-once"

o Select: "Close CD"

-Insert blank CD-R media

-Select OK
