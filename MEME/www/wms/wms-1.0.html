Worklist Management System (WMS)

The WMS is a Web-based system to track and manage the editing process for the UMLS Metathesaurus. The WMS allows different classes of users to examine and modify data relating to worklists. The unauthenticated user is allowed read-only access to the worklist information.

Selecting the information icon: while using the WMS provides context sensitive help.

Read-only Access

  • Query worklist data (User IDs: no authentication necessary)
  • Custom Queries (User IDs: no authentication necessary)
  • Plots (User IDs: no authentication necessary)

Read/Write Access

  • Query and update worklist data (User IDs: nlmmgr, editor)
  • Stamp worklists (User IDs: nlmmgr)
  • Prestamp worklists (User IDs: nlmmgr)
  • Refresh worklist information from system catalogs (User IDs: nlmmgr)

Administrative Access

  • Clean up stale state files (User IDs: nlmmgr)
  • Delete record for worklist: from worklist_info
