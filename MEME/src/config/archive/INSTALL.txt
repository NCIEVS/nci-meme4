INSTALL
-------

2. Unzip dist_archive*.zip into $ARCHIVE_HOME

-----------------------
Subsystem Dependencies
  $ENV_HOME
  $MIDSVCS_HOME

Network Dependencies

  DBI
  DBD::Oracle
