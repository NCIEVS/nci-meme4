README.TXT for MEME Mappings
Last Updated 2011/04/28
Maintained by Joanne Wong (joanne.f.wong@lmco.com)

This directory contains mappings projects (via the NCI UMLS) requested by various parties.

Currently includes:
NCIt to ICD9CM
NCIt to SNOMEDCT
RADLEX to SNOMEDCT
SNOMEDCT to ICD10CM

Directory structure:

README.TXT -- This file
bin/       -- Any scripts used to generate/output mappings
etc/       -- Config files, etc.
data/      -- Any generated data, usually in MRMAP.RRF format
doc/       -- Documentation
